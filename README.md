## DeepMehta 4 Sendgrid

This DeepaMehta 4 Plugin is a wrapper to the [Sendgrid Web API v3](https://sendgrid.com/docs/API_Reference/Web_API_v3/index.html) and exposes core features of the API to other plugins.
